# VISION v2

## THE OBJECTIVE
Transform raw meeting noise into high-clarity tactical notes. Build "Time-to-Think."

## ORGANIC PRINCIPLES
- **Warmth**: Sandstone/Charcoal aesthetics.
- **Dense**: Signal > Noise.
- **Natural**: No developer jargon (e.g., "Ingest" or "Sync"). Use human-centric terms like "Import" and "Activity".

## THE UI LAYOUT
**Dashboard as a Tactical Command Center.**
- **Top 30%**: Intake.
- **Bottom 70%**: Aggregated Action Matrix.
- **Constraints**: No empty states or dead utility pages.
